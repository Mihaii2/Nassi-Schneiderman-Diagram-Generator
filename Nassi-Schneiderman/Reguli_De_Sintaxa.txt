  Regulile de sintaxa ale limbajului Pseudocod utilizate
de catre aceasta aplicatie sunt urmatoarele:

Asignare:        ////<-////

Structura If :   if(////) then
                 {
                     ////
                 }
                 else
                 {
                     /////
                 }

Structura While: while(//////) do 
                 {
                     /////
                 }

Structura For:   for //// <- //// to/down to //// do
                 {
                     /////
                 }

Citire :         read(////)

Afisare:         print(////)

Return:          return(////)

  Aceste reguli respecta urmatoarele conditii:

-Dupa fiecare cuvant cheie urmeaza un set de paranteze, cu 
exceptia sctructurii de control For.

-Nu exista spatiu intre cuvantul cheie si setul de paranteze.

-Instructiunile trebuie pozitionate una sub alta.

-Nu conteaza numarul de spatii inainte de cuvantul
cheie al instructiunii.

-Variabilele nu pot incepe cu literele "for",
de exemplu: formula<-formula + 1
